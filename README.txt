How to use this code:
-This code was made it to plot the Bollinger Bands in the timeframe of 1 minute. The stock I decided to use in this example was BTCUSDT.
-The broker and the information comes from Binance. It refresh every minute and the data continues to reload every 60 seconds.
-In order to use this code, you have to introduce your APIs in the exchange and once you start it, I will keep working until you stop it.
-You can change the stock or crypto in the code by changing the variable "market" and introducing yours.
-This code its not one strategy. This code its only one personal investigation.
-You can change the period of the Bollinger Bands by changing the period variable.
